About the collection

These are compatibility and preservation experiments—not game distributions. Static recompilation translates original game executables for modern targets, while decompilation and reverse-engineering research helps document how the software works. You bring your own legally obtained game and system files; the projects provide the runtime and quality-of-life layer. Pepsiman Recompiled uses PSXRecomp.
