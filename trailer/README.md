## X cut — 32 seconds

The short cut keeps the hook, widescreen reveal, QoL montage, gameplay climax,
and a three-second release card. Both exports are 16:9 so gameplay remains large
and readable in the timeline.

## Audio policy

Capture is silent. The finished edit uses an original synthesized electronic
bed with no captured game audio. The target is -16 LUFS integrated with a
-1.5 dB true-peak ceiling. No copyrighted level music is used.

## Local files

Raw captures and encoded output are intentionally gitignored. The reproducible
edit recipe and release copy remain versioned in this repository.
